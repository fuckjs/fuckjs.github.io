hexo常用markdown语法(基础篇)

1. 标题

#的个数代表几级标题

这是一级标题,1个”#”

这是二级标题,2个”#”

这是三级标题,3个”#”

这是五级标题,4个”#”

注意：#与标题之间一定要有空格。引用结束后要有一空行。
书写如下

## 这是二级标题

2. 换行

两个空格加回车

3. 重新开始一个段落

两个回车

4. 引用

使用”>”即可.引用的效果如下:

这是一段引用

书写如下

> 这是一段引用

5. 代码区块

在代码块前面添加4个空格或者两个tab。效果如下。

function HelloWorld(){
    alert("Hello World");
}

6. 图片引用

;
书写如下

>![百度](https://ss0.bdstatic.com/5aV1bjqh_Q23odCf/static/superman/img/logo/bd_logo1_31bdc765.png);

7. 其他

不同的markdown模板引擎，markdown语法大体相同，在细微处有些差别。初学可以参考作业部落
