  • 2016-01
  • jQuery
前端开发

一. jQuery是如何向外提供接口的?

jQuery.fn = jQuery.prototype = function(){

}

例如: $("#div1).css() $()是上方构造函数包装好的一个方法，返回值是一个jQuery对象。因此可以调用jQuery原型上的相关方法。

二. 最外层的匿名自执行函数为什么传window和undefined?

之所以传window作为参数，有两个原因。

  • 因为window是作用域最顶端，查找速度比较慢
  • 在压缩的时候有用

而之所以传undefined，是因为undefined是可以被人为修改的。这样在内部就可以保证undefiend没有被修改。
